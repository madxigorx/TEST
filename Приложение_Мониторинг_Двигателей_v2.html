📘 Инструкция по использованию

🎯 Назначение приложения

Веб-приложение для учёта электродвигателей в ремонте: позволяет добавлять двигатели, отслеживать их статус, помечать важные для протокола ЦЭО и автоматически сохранять базу данных в файл.

🚀 Быстрый старт

Шаг 1: Откройте приложение

Откройте HTML файл в браузере. Рекомендуемые браузеры: Chrome, Edge или Яндекс Браузер.

Шаг 2: Настройте автосохранение

  1. Нажмите кнопку "📁 Выбрать папку для автосохранения"
  2. Выберите удобную папку (например, Документы\Базы данных\)
  3. Браузер попросит разрешение — нажмите "Разрешить при каждом посещении" или "View files" → "Save changes"
  4. Готово! Файл Двигатели_БД.json будет автоматически обновляться
💡 Совет: Статус автосохранения отображается зелёным цветом с названием выбранной папки.

📝 Основные функции

1️⃣ Добавление

Заполните форму и добавьте двигатель в базу. Обязательные поля: участок, наименование, номер, мощность.

2️⃣ В ремонте

Просмотр всех двигателей в ремонте. Можно искать, фильтровать по периоду и экспортировать в Word.

3️⃣ Готовые

Двигатели, вышедшие из ремонта. Помечайте готовыми через кнопку "✓ Готов" во вкладке "В ремонте".

4️⃣ Протокол ЦЭО

Избранные двигатели для протокола. Помечайте звёздочкой "☆ ЦЭО" в любой вкладке.

🔍 Поиск и фильтры

Умный поиск понимает:

  • Числа — ищет в мощности и оборотах: 11 найдёт "11 кВт" и "1100 об/мин"
  • Участки — А1, ЦЭО, участок 5
  • Тип ремонта — кап (капитальный), сред (средний)
  • Несколько слов — А1 11 1500 найдёт двигатели на участке А1, 11 кВт, 1500 об/мин

Примеры поиска:

Запрос Результат
11 Все двигатели 11 кВт
А1 Все двигатели участка А1
кап Все капитальные ремонты
А1 11 кап Двигатели А1, 11 кВт, капремонт

Фильтр по периоду:

Укажите "Период с" и "Период по" для фильтрации двигателей, ремонт которых происходил в этом промежутке. Работает только для записей с заполненными датами ремонта и выхода.

✏️ Редактирование и удаление

Редактирование:

  1. Нажмите "✎ Изменить" на нужной записи
  2. Откроется вкладка "Добавление" с заполненными полями
  3. Измените данные и нажмите "Сохранить изменения"
  4. Или нажмите "Отменить редактирование" для отмены

Удаление:

Нажмите "✕ Удалить" на записи. Удаление происходит мгновенно и необратимо.

💾 Сохранение данных

Автосохранение (рекомендуется):

  • После выбора папки данные сохраняются при каждом изменении
  • Файл: Двигатели_БД.json (перезаписывается)
  • Работает после перезагрузки страницы

Ручной экспорт:

  • Кнопка "💾 Экспорт базы в файл" создаёт файл с датой
  • Скачивается в папку "Загрузки"
  • Используйте для резервных копий

Импорт данных:

  • Кнопка "📂 Импорт базы из файла" восстанавливает данные
  • Выберите ранее экспортированный .json файл
⚠️ Внимание: Импорт заменяет ВСЕ текущие данные на данные из файла! Сделайте экспорт перед импортом для сохранности.

📊 Экспорт в Word

Кнопка "Экспорт в Word" доступна во вкладках "В ремонте", "Готовые" и "Протокол ЦЭО".

Что экспортируется:

  • Все записи, которые видны после применения фильтров
  • Если поиск или период активны — только отфильтрованные данные
  • Формат: Microsoft Word (.doc), альбомная ориентация A4

⚠️ Важные ограничения

Автосохранение работает в:

  • ✅ Chrome 86+, Edge 86+, Яндекс Браузер, Opera
  • ❌ Firefox, Safari, мобильные браузеры

Данные хранятся:

  1. В браузере (localStorage) — автоматически
  2. В файле (если настроено) — автоматически
  3. В Downloads (при ручном экспорте) — по кнопке
⚠️ Внимание: Разрешения на доступ к папке могут сбрасываться при обновлении браузера — выберите папку заново.

🆘 Решение проблем

"Автосохранение не работает"

  1. Проверьте браузер — используйте Chrome/Edge/Яндекс
  2. Нажмите "Выбрать папку" заново
  3. Разрешите доступ к файлам в диалоге браузера

"Данные потерялись"

  1. Перезагрузите страницу — данные в localStorage восстановятся
  2. Проверьте папку автосохранения — файл Двигатели_БД.json
  3. Проверьте папку "Загрузки" — резервные копии с датой

"Кнопка '✓ Готов' не активна"

Заполните поле "Дата выхода из ремонта" перед тем, как помечать готовым. Нажмите "✎ Изменить", добавьте дату и сохраните.

💡 Рекомендации

Для начала работы:
  1. Настройте автосохранение (выберите папку)
  2. Добавьте 2-3 тестовых записи
  3. Попробуйте поиск и фильтры
  4. Сделайте экспорт в Word
Для ежедневной работы:
  • Добавляйте новые двигатели через вкладку "Добавление"
  • Отмечайте готовые кнопкой "✓ Готов"
  • Помечайте важные звёздочкой "☆ ЦЭО"
  • Используйте поиск для быстрого нахождения
Резервное копирование:

Раз в неделю делайте ручной экспорт кнопкой "💾 Экспорт базы в файл". Храните файлы в безопасном месте (облако, флешка).

🎨 Дополнительно

Переключение темы: кнопка 🌙 Тёмная / ☀️ Светлая в правом верхнем углу. Тема сохраняется автоматически.
